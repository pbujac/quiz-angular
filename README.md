Quiz UI App - Angular
===========
Quiz client website
## Built With

* [Angular 2](https://symfony.com) -  The web framework used
* [Sass](http://sass-lang.com) - CSS Style preprocessor
* [Foundationa](http://sass-lang.com) - CSS Style preprocessor

## Run project
Make sure you have installed 
- Docker, 
- NodeJS, 
- Angular CLI

 Open project quiz-php and run in terminal:
```
docker-compose build
docker-compose up
```

 Open project quiz-angular(current) and run in terminal:
```
npm install
ng serve
```
